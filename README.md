# deck-of-cards
by [Scott Macpherson](https://github.com/scottmacphersonmusic)
#### Description
A standard deck of 52 cards that can be shuffled and dealt.
#### Usage
Create a new deck:

`deck = Deck.new`

Shuffle those cards, you cheater!:

`deck.shuffle!`

Deal a card:

`deck.deal_card`

Never reveal your tell.
